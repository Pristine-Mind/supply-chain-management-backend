üìã API Endpoints

POST /api/public/external-delivery/register/

Register External Business - Submit your business for approval

GET /api/public/external-delivery/track/{tracking_number}/

Public Tracking - Track any delivery using tracking number (no auth required)

POST /api/external/deliveries/

Create Delivery - Create a new delivery request

GET /api/external/deliveries/

List Deliveries - Get all your deliveries with filtering options

GET /api/external/deliveries/{id}/

Delivery Details - Get detailed information about a specific delivery

POST /api/external/deliveries/{id}/cancel/

Cancel Delivery - Cancel a delivery (if status allows)

GET /api/external/dashboard/

Business Dashboard - Get comprehensive business statistics

POST /api/external/webhook/test/

Test Webhook - Test your webhook endpoint
