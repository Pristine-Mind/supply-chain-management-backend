## 🚀 Quick Start

### Step 1: Navigate to Admin
```
1. Open: https://yoursite.com/admin/
2. Login with superuser credentials
3. Go to: Authentication and Authorization → Business Users
```

### Step 2: Select & Export
```
1. Check the checkbox for the business user(s)
2. Select: 📊 Export Business Data to Excel
3. Click: Go
4. Download will start automatically
```

### Step 3: Open & Analyze
```
1. Open Excel file: Business_Data_[username]_[shop_id].xlsx
2. View Executive Summary first
3. Explore other sheets for detailed data
4. Use Key Metrics as reference guide
```

## 📞 Troubleshooting Quick Reference

| Issue | Solution |
|-------|----------|
| No sheets appear | Ensure business user has data |
| Data missing | Check if data is linked to user |
| Slow to generate | Dataset is large, wait 5-10 seconds |
| Formatting broken | Update to Excel 2016 or later |
| Export not showing | Clear browser cache, try again |
